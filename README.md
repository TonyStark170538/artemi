Artemis II — Interactive Storytelling Website

A cinematic scrolling web experience that tells the story of NASA's Artemis II mission through immersive visuals, motion, sound, and interactive storytelling.

📌 Project Overview

Artemis II is a modern storytelling website designed to educate and inspire users about humanity's return to deep space exploration.

Rather than presenting information as a traditional website, this project transforms the mission into an interactive digital journey—where users scroll through the launch, spacecraft systems, crew introduction, mission trajectory, Moon flyby, and future lunar exploration.

The experience combines:

Cinematic storytelling
High-quality visuals and NASA media
Interactive animations
Immersive UI/UX
Scroll-driven narrative progression

The goal is to make visitors feel like they are part of the mission.
